# advent-of-code-2023

usage:
py main.py $day [-t] [-d]

parameters:
$day: mandatory, the day number (1,2... 25)

options:
-t: uses test cases (eg. py main.py 1 -t test_case_1). Test cases are stored in data/dayXY/tests/
-d: enable debug messages
